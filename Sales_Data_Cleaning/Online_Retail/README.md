\# Sales Data Cleaning using Pandas



\## Objective



Perform data cleaning on the Online Retail Sales dataset using Pandas functions.



\## Dataset



\- Dataset Name: Online Retail Dataset

\- Source: Kaggle



\## Tasks Performed



\- Imported Pandas library

\- Loaded the dataset

\- Displayed first and last records

\- Checked dataset shape

\- Displayed dataset information

\- Checked column names

\- Checked data types

\- Generated statistical summary

\- Identified missing values

\- Removed duplicate records

\- Filled missing values

\- Cleaned column names

\- Converted InvoiceDate to datetime

\- Counted unique values

\- Saved the cleaned dataset



\## Libraries Used



\- Pandas



\## Files Included



\- online\_retail.csv

\- Cleaned\_Online\_Retail.csv

\- Sales\_Data\_Cleaning.ipynb

\- README.md



\## Learning Outcome



This project demonstrates how to clean a real-world sales dataset using Pandas by handling missing values, removing duplicate records, correcting data types, and preparing the dataset for further analysis.
