# SQL Assignment

## Objective

This assignment demonstrates fundamental SQL concepts using the **Employee** and **Department** tables.

## Topics Covered

1. Top 5 Highest Salary Employees
2. Department-wise Employee Count
3. Second Highest Salary
4. Employees Whose Salary is Greater Than Department Average
5. Inner Join
6. Left Join
7. GROUP BY with HAVING
8. Employees Hired in the Last 6 Months
9. Find Duplicate Records
10. Remove Duplicate Records

## Tools Used

* MySQL Workbench
* Git
* GitHub

## Files Included

* `SQL_Internship_Task.sql` – SQL script containing table creation, sample data, and all queries.
* Output screenshots for each query execution.
* `README.md` – Documentation for the assignment.

## Learning Outcomes

* Creating and managing tables
* Inserting records
* Writing SQL queries
* Using JOINs
* Applying GROUP BY and HAVING
* Working with aggregate functions
* Identifying and removing duplicate records
* Uploading SQL projects to GitHub
