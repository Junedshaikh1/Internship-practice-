# SQL Internship Tasks

## Objective

This repository contains SQL internship tasks completed using **MySQL Workbench**. The assignments cover both fundamental and advanced SQL concepts, helping build practical database management and querying skills.

---

## Task 1: Basic SQL Queries

### Topics Covered

1. Top 5 Highest Salary Employees
2. Department-wise Employee Count
3. Second Highest Salary
4. Employees Whose Salary is Greater Than Department Average
5. Inner Join
6. Left Join
7. GROUP BY with HAVING
8. Employees Hired in the Last 6 Months
9. Find Duplicate Records
10. Remove Duplicate Records

### Files Included

* `SQL_Internship_Task.sql` – Basic SQL queries with table creation, sample data, and solutions.
* `SQL_Assignment_Report.pdf` – Output screenshots and query results.

---

## Task 2: Advanced SQL Concepts

### Topics Covered

1. Add Data to Existing Tables
2. Window Functions (`ROW_NUMBER()`, `RANK()`, `DENSE_RANK()`)
3. CASE WHEN Statement
4. Views
5. Triggers

### Files Included

* `SQL_Internship_Advance_Task.sql` – Advanced SQL queries.
* `Advanced_SQL_Assignment_Report.pdf` – Output screenshots and results.
